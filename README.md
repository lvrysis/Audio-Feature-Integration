# Audio Feature Integrator

https://scholar.google.gr/scholar?cluster=8786410930324265097  
https://scholar.google.gr/scholar?cluster=8786410930324265097  
https://scholar.google.gr/scholar?cluster=8786410930324265097  

The implementations are powered by Excel+VBA and MATLAB.
